# SecureAuth - JWT Authentication System
A production-ready authentication system built using Django REST Framework with JWT based authentication, custom user model,PostgreSQL database integration.
The project provides secure user registration, login functionality, and token-based authentication using a custom user model

# Tech Stack
Backend:
- Python
- Django
- Django REST Framework

Authentication:
- JWT (Simple JWT)
- Custom User Model

Database:
- PostgreSQL

Tools:
- Git
- Docker

# 🛡 Features
- Passwords are stored using Django password hashing
- login api with JWT based authentication
- Custom user authentication system
- Environment variables for sensitive information
- Email-based authentication
- PostgreSQL Database Integration
- Role and Permission Support
- Secure API Authentication
- Deploy using AWS

# Intstallation and Setup
1- clone repo from remote(github)
2- create virtual environment
3- install dependencies(requirements.txt)
4- create .env file (environment variables)
5- database setup(postgresql)
6- run migrations 
7- create admin user(superuser) then run server
8- create register api by DRF(serializers-views-urls)
9- create login api with token generated
